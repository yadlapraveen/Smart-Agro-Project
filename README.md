# Smart Agro

A React + Vite e-commerce marketplace prototype. All data is stored locally within the browser; there is no backend or database.

## Running the project

1. Ensure Node.js (v18+) is installed.
2. In the project root run:
   ```bash
   npm install
   npm run dev
   ```
3. Open http://localhost:5174 in your browser to use the app.

The source includes a `sampleProducts` array and the application state is persisted to `localStorage`. Modify the data in `src/data/products.js` to customize the product catalogue.

No server, database or API calls are made.
